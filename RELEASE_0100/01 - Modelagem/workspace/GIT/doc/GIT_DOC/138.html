UC059 - Visualizar Painel de Controle
UseCase
Fluxo Principal - Visualizar Painel de Controle FP.1 - O usuário solicita visualizar o Painel de Controle. FP.2 - O sistema apresenta os dados do último Processamento realizado (FA1, INF-01). FP.3 - O sistema apresenta os Widgets do Perfil do usuário logado (INF-02). FP.4 - O sistema encerra o caso de uso. Fluxos Alternativos FA1 - Processar Atualização FA1.1 - No passo FP.2, o usuário solicita realizar um processamento. FA1.2 - O sistema executa o processamento de acordo com o P1 [ UC054 - Processar Atualização ]. FA1.3 - O sistema inclui as informações do Processamento (INF-03). FA1.4 - O sistema retorna ao passo FP.2. Informações Complementares INF-01 - Informações Apresentadas do Último Processamento Realizado INF-02 - Informações Apresentadas dos Widgets do Perfil INF-03 - Informações Incluídas do Processamento UCs Relacionados P1 [ UC054 - Processar Atualização ]
Exibe os widgets configurados par ao perfil de usuário que está acessando o sistema.
